# YouTube Resources

Curated YouTube channels and playlists for cybersecurity and ethical hacking.

> All resources listed here are freely available on YouTube.

---

## Primary Course

### freeCodeCamp — Learn Cybersecurity and Ethical Hacking Using Kali Linux
- **URL:** https://youtu.be/ug8W0sFiVJo
- **Duration:** ~4 hours
- **Instructor:** Sunny Dimalu "The Cyborg"
- **Topics:** Terminal basics, Nmap, Aircrack-ng, WPA2 cracking, Wireshark
- **Best for:** Absolute beginners, first month of study plan

## Recommended Channels

### TCM Security (Heath Adams)
- **Channel:** https://www.youtube.com/@TCMSecurityAcademy
- Practical ethical hacking, Active Directory attacks, OSCP prep
- Free "Practical Ethical Hacking" course excerpts
- **Recommended videos:**
  - "Ethical Hacking in 15 Hours – Full Course"
  - "Active Directory Attacks"

### IppSec
- **Channel:** https://www.youtube.com/@ippsec
- Detailed HackTheBox machine walkthroughs (after machines retire)
- Shows real methodology: enumeration → foothold → privesc
- **Best used:** After attempting HTB machines yourself

### John Hammond
- **Channel:** https://www.youtube.com/@_JohnHammond
- CTF walkthroughs, malware analysis, threat hunting
- Great for forensics and CTF skills

### NetworkChuck
- **Channel:** https://www.youtube.com/@NetworkChuck

### The Cyber Mentor (TCM)

## Tips for Using YouTube Effectively

1. **Code along** — don't just watch, run the same commands in your Docker lab
2. **Use timestamps** — most long tutorials have chapters, jump to relevant sections
3. **Pause and practice** — after each technique, try it in your lab before moving on
4. **Watch IppSec** only after attempting a box yourself — spoilers reduce learning
5. **Speed it up** — 1.5x speed is fine for review, use 1x when learning new concepts
