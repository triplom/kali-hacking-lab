# freeCodeCamp Resources

All freeCodeCamp resources are **free** and beginner-friendly. The main course covers core Kali tools; the articles provide deep dives into specific topics.

---

## Main Course — Learn Cybersecurity and Ethical Hacking Using Kali Linux

**URL:** https://www.freecodecamp.org/news/learn-cybersecurity-and-ethical-hacking-using-kali-linux/
**YouTube:** https://youtu.be/ug8W0sFiVJo
**Duration:** ~4 hours
**Instructor:** Sunny Dimalu "The Cyborg"

### Course Sections & Study Plan Mapping

| Section | Topics | Study Plan Month |
|---------|--------|-----------------|
| Basic Commands & Terminal | ls, cd, cat, chmod, sudo, file system navigation | Prerequisites (Week 0) |
| Nmap / Scan Ports | Host discovery, port scanning, service detection, scan types | Month 1–2: Information Gathering |
| Monitor Mode | Wireless interface modes, airmon-ng | Month 6: Wireless |
| Aircrack-ng | WPA2 capture and cracking workflow | Month 6: Wireless |
| 4-Way Handshake | How WPA2 authentication works | Month 6: Wireless |
| De-auth Attacks | aireplay-ng deauthentication, forcing handshake | Month 6: Wireless |
| Wordlists | rockyou.txt, custom wordlists, CeWL | Month 5: Password Attacks |
| WiFi Cracking | End-to-end WPA2 crack with aircrack-ng | Month 6: Wireless |
| Wireshark Tutorial | Packet capture, filters, TCP stream analysis | Month 8: Sniffing |

### Key Takeaways from the Course
- Great for absolute beginners — starts from the command line
- Wireless section is the most comprehensive part (covers Months 6 entirely)
- Wireshark section pairs well with the sniffing and spoofing tools reference
- Nmap section is a solid intro before moving into the full study plan Month 1–2

## How to Use These Resources

1. **Watch the main course first** as a broad overview before your first month
2. **Refer to the articles** when you reach the corresponding month in the study plan
3. **Combine with hands-on Docker lab** — read article, then practice in lab
4. **Use freeCodeCamp's responsive design certification** as a bonus if you want to understand web targets better
